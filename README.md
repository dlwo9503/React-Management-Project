# React-Management-Project

https://dlwo9503.github.io/React-Management-Project/

고객 관리 시스템 구현
 - 고객 추가 기능 (이미지, 이름, 생년월일, 성별, 직업)
 - 고객 삭제 기능
 - 고객 이름 검색 기능

사용 기술
 - React.js
 - node.js
 - MySQL
 - CSS (Material UI)
